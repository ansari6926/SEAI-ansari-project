PROJECT SUMMARY


RoadSense AI is a complete SEAI mini project built around road damage detection. It uses a FastAPI backend, browser frontend, ONNX-ready inference flow, Docker configuration, Kubernetes deployment file, Git version control, and a Software Impacts-style paper.


NOVELTY SUMMARY


The main novelty is the Civic Repair Planner. It turns road damage detections into repair cost, crew size, SLA hours, lane closure requirement, risk zone, SDG alignment, and dispatch note. This makes the system more useful than a simple object detection demo.
